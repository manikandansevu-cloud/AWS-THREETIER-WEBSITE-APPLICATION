# AWS Three-Tier Web Application Deployment

## Project Overview
This project demonstrates the deployment of a scalable and highly available Three-Tier Web Application on AWS.

## AWS Services Used
- Amazon EC2
- Amazon RDS
- Application Load Balancer (ALB)
- Auto Scaling Group
- Amazon VPC
- Security Groups

## Architecture
### Presentation Tier
- Hosts the web application on EC2 instances.
- Receives user traffic through the Load Balancer.

### Application Tier
- Processes application requests and business logic.

### Database Tier
- Stores application data in Amazon RDS.

## Features
- High Availability
- Scalability
- Secure Network Design
- Load Balancing
- Database Integration

## Outcome
Successfully deployed a Three-Tier Architecture using AWS cloud services.
